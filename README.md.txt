Gestor de Citas y Agenda

Este proyecto es un sistema básico en Java que permite gestionar citas y agendas de manera sencilla. Se realizó aplicando los principios de la programación orientada a objetos (POO), con clases, herencia, relaciones, excepciones y un menú principal para interactuar con el usuario.

Descripción

El programa simula la gestión de citas en una agenda. Permite registrar personas (como pacientes o profesionales), crear citas, listarlas y cancelarlas. Se buscó que fuera fácil de entender y ampliar en el futuro.

Estructura del proyecto

El código está organizado en clases:

Persona: Clase base que contiene los datos generales de una persona.

Paciente y Profesional: Heredan de Persona y agregan atributos propios.

Cita: Representa una cita entre un paciente y un profesional, con fecha y hora.

Agenda: Administra las citas (agregar, listar, eliminar).

GestorCitas: Contiene el menú principal y gestiona la interacción con el usuario.

Requerimientos funcionales

Registrar pacientes y profesionales.

Asignar citas con fecha y hora.

Listar todas las citas registradas.

Eliminar o cancelar una cita.

Validar datos de entrada para evitar errores.
